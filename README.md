# 📚 Library

Modern book search application built with React, TypeScript, and OpenLibrary API.

---

# 🚀 About Project

Library is a frontend application for searching and browsing books using the OpenLibrary API.

The project was created to practice scalable frontend architecture, API integration, and modern React development approaches.

---

# 🛠 Tech Stack

## Frontend
- React
- TypeScript
- Vite

## State & Data
- Fetch API
- OpenLibrary API

## Architecture
- Feature-Sliced Design (FSD)

## Styling
- Tailwind
- CSS3
- Responsive Layout

---

# ✨ Features

- 🔍 Book search
- 📖 Book information display
- ⚡ Fast Vite build
- 🧩 Modular architecture
- 📱 Responsive UI
- ♻️ Reusable components

---

# 📂 Project Structure

```bash
src/
│
├── app/          # App configuration
├── pages/        # Application pages
├── widgets/      # Large UI blocks
├── features/     # Business features
├── entities/     # Entities (book, etc.)
├── shared/       # Shared components/utilities
│
└── main.tsx
````
